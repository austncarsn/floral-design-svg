# Floral Design System

An elegant digital collection of **44 geometric floral SVGs**, crafted as a cohesive design system for modern web and branding use. Each element balances mathematical precision with organic symmetry, offering scalable vector assets ideal for UI, print, and motion applications.

## Features

- **44 Geometric Designs** — a curated set of modular floral and stem components  
- **Gallery View** — browse all designs in a clean, responsive interface  
- **Color Variations** — explore tonal palettes and visual treatments  
- **Vector Downloads** — instantly export SVGs for your creative workflow  
- **Refined Aesthetic** — minimal gradients, soft contrast, and timeless composition  

## Technology

- Built with **React + Vite + TailwindCSS**  
- SVG system optimized for **accessibility, scalability, and design consistency**  
- Supports light and dark themes with smooth transitions  

## Use Cases

Perfect for:
- Branding systems  
- Editorial graphics  
- Pattern and textile design  
- UI iconography  
- Motion studies
